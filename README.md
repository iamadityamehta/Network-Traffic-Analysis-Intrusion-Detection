# 🚨 Network Traffic Analysis & Intrusion Detection

## 📌 Introduction
This project focuses on analyzing network traffic and implementing basic **Intrusion Detection techniques** using open-source tools. The goal is to monitor network activity, detect suspicious behavior, and understand how potential attacks can be identified in real-world scenarios.

---

## 🎯 Objective
- To analyze network traffic in real-time  
- To detect suspicious or malicious activities  
- To understand intrusion detection concepts  
- To enhance network security awareness  

---

## 📍 Scope
- Packet capturing and analysis  
- Detection of abnormal traffic patterns  
- Basic intrusion detection techniques  
- Understanding attack indicators  

---

## 🛠️ Tools Used
- Wireshark (Packet Analysis)  
- Snort (Intrusion Detection System) *(if used)*  
- Kali Linux / Linux Environment  

---

## ⚙️ Methodology

1. **Environment Setup**
   - Installed and configured monitoring tools  
   - Prepared system for packet capture  

2. **Traffic Capture**
   - Captured live network traffic  
   - Stored packets for detailed inspection  

3. **Traffic Analysis**
   - Analyzed protocols (TCP, UDP, HTTP, DNS)  
   - Applied filters to isolate relevant data  

4. **Intrusion Detection**
   - Identified unusual patterns  
   - Detected possible malicious activities  
   - Observed indicators of compromise (IoCs)  

5. **Reporting**
   - Documented findings  
   - Provided insights and observations  

---

## 📊 Key Findings

- Observed protocol-level behavior useful for intrusion detection
